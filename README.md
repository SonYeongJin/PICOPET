# 🐾 wag — 반려견 심박수 모니터링 앱

> 반려견의 심박수를 실시간으로 확인하고 건강을 관리하는 IoT 앱

---

## 소개

반려견의 지속 가능한 라이프스타일 향상을 목표로 개발한 프로젝트입니다.  
심박수 측정 센서와 Flutter 앱이 BLE로 연동되어,  
보호자가 반려견의 심박수와 건강 상태를 실시간으로 확인할 수 있습니다.

한남대학교 **Innovation Product Development(IPD)** 수업에서  
TJB 기업 협업으로 진행된 팀 프로젝트입니다.

---

## 주요 기능

- 반려견 정보 등록 및 관리
- BLE 기반 심박수 센서 실시간 연동
- 심박수 모니터링 및 건강 이상 알림
- 위치 기반 가까운 동물병원 안내

---

## 사용 기술

`Flutter` `Arduino` `BLE` `Android` `iOS`

---

## 담당 역할
- Flutter 앱 전체 개발
- Arduino 기반 심박수 센서 하드웨어 개발
- BLE 통신 연동
